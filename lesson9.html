Lesson 9: Mining Algorithms and How to Mine Each

The Veil network supports three Proof-of-Work mining algorithms. Each is optimized for a different type of hardware, offering flexibility for miners and greater decentralization.

  • ProgPoW: GPU-friendly and ASIC-resistant. Use miners like T-Rex, TTminer, or Wildrig. Requires the solo mining proxy.
  • RandomX: CPU-optimized. Use XMRig compiled from source. Source code is available here.
  • SHA256d: The original Bitcoin algorithm. Currently, SHA256d mining is only available inside the Veil Core Wallet using in-wallet CPU mining. A guide is available here.

Each algorithm runs on its own chain with independent difficulty. Choose based on your available hardware. ProgPoW is the most active mining option. RandomX is supported by the only public mining pool, available here.
