Lesson 9: Mining Algorithms and How to Mine Each

The Veil network supports three Proof-of-Work mining algorithms. Each is optimized for a different type of hardware, offering flexibility for miners and greater decentralization.

  • ProgPoW: GPU-friendly and ASIC-resistant. Use miners like T-Rex, TTminer, or Wildrig. Requires the solo mining proxy.
  • RandomX: CPU-optimized. Use XMRig compiled from source: xmrig-veil.
  • SHA256d: The original Bitcoin algorithm. Currently available only via in-wallet CPU mining in the Veil Core Wallet. Guide: Veil in-wallet mining guide .

Each algorithm has its own difficulty and reward schedule. Choose based on the hardware you have. ProgPoW is typically the most active, while RandomX mining is supported by the public pool Fastpool.
