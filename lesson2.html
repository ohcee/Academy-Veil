Lesson 2: Setting Up the Veil Wallet & Using a Snapshot

Before you can stake, send, or mint Veil, you will need to set up the Veil Core Wallet. This lesson will guide you through:

  • Downloading and installing the wallet
  • Using a blockchain snapshot to sync faster
  • Basic wallet setup tips

Install the Wallet

Go to Veil Releases and download the wallet for your OS.
Install it like any other app. On first run, close the wallet so you can use the snapshot method below.

Download and Apply the Snapshot

  1. Go to the Veil Tools site and download the snapshot closest to your region.
  2. Extract the snapshot. (Example: to your Desktop.)
  3. Open your file browser and navigate to your home folder.
  4. Find and open the .veil folder.
  5. Delete the following folders inside .veil:
    • blocks
    • chainstate
    • indexes
    • zerocoin
  6. Copy or move the same folders from the snapshot into the .veil folder.
  7. Once transferred, close the window and launch the wallet.

First Wallet Steps

  • Let it fully sync, then you're ready to mint and stake
  • (Sugguested) Encrypt your wallet from Settings

1. What folders must you delete before applying the snapshot?

wallet.dat and peers.dat
blocks, chainstate, indexes, zerocoin
logs and database
